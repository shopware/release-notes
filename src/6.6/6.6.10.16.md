# Release notes Shopware 6.6.10.16

## Abstract

This is a patch release containing bug fixes and a few backported improvements across CMS, admin UI, cart/promotions, search, performance, and stability.

## System requirements

* tested on PHP 8.2 and 8.4
* tested on MySQL 8 and MariaDB 11

## Improvements

(No notable improvements in this patch release)
