# Release notes Shopware 6.7.9.1

## Abstract

This patch release for Shopware 6.7.9.1 contains only bug fixes. It addresses five issues, including a broken desktop layout, migration errors, reCAPTCHA validation restoration, and a quantity range constraint.

## System requirements

* tested on PHP 8.2, 8.4 and 8.5
* tested on MySQL 8 and MariaDB 11

## Improvements

(No notable improvements in this patch release)

## Fixed bugs

* [16298](https://github.com/shopware/shopware/issues/16298) Sw-Desktop layout broken
* [16378](https://github.com/shopware/shopware/pull/16378) non existence default category layout when migrating
* [16403](https://github.com/shopware/shopware/pull/16403) restore removal of required validator from reCAPTCHA inputs
* [15584](https://github.com/shopware/shopware/issues/15584) Migration1768545320RevocationRequestCmsForm causes duplicate entry if de-DE or en-GB is missing
* [16421](https://github.com/shopware/shopware/pull/16421) only require min value of quantity range with major version 6.8
