# Release notes Shopware 6.7.8.2

## Abstract

This patch release fixes at least three bugs with critical/high priority.

## System requirements

* tested on PHP 8.2, 8.4 and 8.5
* tested on MySQL 8 and MariaDB 11

## Improvements

(No notable improvements in this patch release)

## Fixed bugs

* [#15520](https://github.com/shopware/shopware/issues/15520) Digital Products download/access not possible
* [#15395](https://github.com/shopware/shopware/issues/15395) Order state change throws error if you have installed app that activates webhooks for order status change
* [#15358](https://github.com/shopware/shopware/issues/15358) Administration: price inheritance is not applied correctly for product variants
