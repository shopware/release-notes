# Release notes Shopware 6.7.1.1

## Abstract

This patch release contains just bug fixes.

## System requirements

* tested on PHP 8.2, 8.3 and 8.4
* tested on MySQL 8 and MariaDB 11

## Improvements

(No notable improvements listed in v6.7.1.1)
