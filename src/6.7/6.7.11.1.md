# Release notes Shopware 6.7.11.1

## Abstract

Shopware 6.7.11.1 is a patch release that fixes three bugs.

## System requirements

* tested on PHP 8.2, 8.4 and 8.5
* tested on MySQL 8 and MariaDB 11

## Improvements

(No notable improvements in this patch release)

## Fixed bugs

* [#16744](https://github.com/shopware/shopware/issues/16744) Nested Footer Nav and/or Footer Service Nav leads to missing Shipping SEO Url in VAT Message
* [#17434](https://github.com/shopware/shopware/issues/17434) theme:compile makes HTTP requests to itself to read bundle build-meta, causing slow & unreliable compilation
* [#17505](https://github.com/shopware/shopware/issues/17505) TypeError: McpAuthenticationListener requires concrete ClientRepository, breaks when service is replaced (e.g. Heptacom AdminOpenAuth)
